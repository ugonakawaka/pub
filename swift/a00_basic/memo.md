### まずはバージョンを確認しましょう
```
swift --version 
```

### コンパイルするよ
```
swiftc main.swift 
swiftc main.swift -suppress-warnings 
```

### 即時実行
```
swift main.swift
```

### swftformt
```
brew install swiftformat
swiftformat main.swift
```
### ワーニングをおさえるよ  
```
-suppress-warnings      Suppress all warnings  
```
